д»Җд№ҲжҳҜ HertzBeat?вҖӢ

HertzBeat иө«е…№и·іеҠЁ жҳҜдёҖдёӘжӢҘжңүејәеӨ§иҮӘе®ҡд№үзӣ‘жҺ§иғҪеҠӣпјҢй«ҳжҖ§иғҪйӣҶзҫӨпјҢе…је®№ PrometheusпјҢж— йңҖ Agent зҡ„ејҖжәҗе®һж—¶зӣ‘жҺ§е‘ҠиӯҰзі»з»ҹгҖӮ

зү№зӮ№вҖӢ

  • йӣҶ зӣ‘жҺ§+е‘ҠиӯҰ+йҖҡзҹҘ дёәдёҖдҪ“пјҢж”ҜжҢҒеҜ№еә”з”ЁжңҚеҠЎпјҢеә”з”ЁзЁӢеәҸпјҢж•°жҚ®еә“пјҢзј“еӯҳпјҢж“ҚдҪңзі»з»ҹпјҢеӨ§ж•°жҚ®пјҢдёӯй—ҙд»¶пјҢWebжңҚеҠЎеҷЁпјҢдә‘еҺҹз”ҹпјҢзҪ‘з»ңпјҢиҮӘе®ҡд№үзӯүзӣ‘жҺ§йҳҲеҖје‘ҠиӯҰйҖҡзҹҘдёҖжӯҘеҲ°дҪҚгҖӮ
  • жҳ“з”ЁеҸӢеҘҪпјҢж— йңҖ AgentпјҢе…Ё WEB йЎөйқўж“ҚдҪңпјҢйј ж ҮзӮ№дёҖзӮ№е°ұиғҪзӣ‘жҺ§е‘ҠиӯҰпјҢйӣ¶дёҠжүӢеӯҰд№ жҲҗжң¬гҖӮ
  • е°Ҷ Http, Jmx, Ssh, Snmp, Jdbc, Prometheus зӯүеҚҸи®®и§„иҢғеҸҜй…ҚзҪ®еҢ–пјҢеҸӘйңҖеңЁжөҸи§ҲеҷЁй…ҚзҪ®зӣ‘жҺ§жЁЎжқҝ YML е°ұиғҪдҪҝз”ЁиҝҷдәӣеҚҸи®®еҺ»иҮӘе®ҡд№үйҮҮйӣҶжғіиҰҒзҡ„жҢҮж ҮгҖӮжӮЁзӣёдҝЎеҸӘйңҖй…ҚзҪ®дёӢе°ұиғҪз«ӢеҲ»йҖӮй…ҚдёҖж¬ҫ K8s жҲ– Docker зӯүж–°зҡ„зӣ‘жҺ§зұ»еһӢеҗ—пјҹ
  • е…је®№ Prometheus зҡ„зі»з»ҹз”ҹжҖҒе№¶дё”жӣҙеӨҡпјҢеҸӘйңҖйЎөйқўж“ҚдҪңе°ұеҸҜд»Ҙзӣ‘жҺ§ Prometheus жүҖиғҪзӣ‘жҺ§зҡ„гҖӮ
  • й«ҳжҖ§иғҪпјҢж”ҜжҢҒеӨҡйҮҮйӣҶеҷЁйӣҶзҫӨжЁӘеҗ‘жү©еұ•пјҢж”ҜжҢҒеӨҡйҡ”зҰ»зҪ‘з»ңзӣ‘жҺ§пјҢдә‘иҫ№еҚҸеҗҢгҖӮ
