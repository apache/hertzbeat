# Intelligent Inspection (AI Workflow)

## Overview
This feature provides daily system health checks using LLM-based analysis.

## Features
- Scan all monitors and active alerts
- Collect trend data for abnormal monitors
- Correlation analysis and risk assessment
- Generate a concise Markdown report

## Benefits
- Reduce alert fatigue
- Proactive risk detection
- Better system overview

## Usage
This feature can be enabled via the hertzbeat-ai module.
