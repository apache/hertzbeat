> 通过配置application.yml文件里面的aiConfig实现与人工智能的对话

### 配置参数说明

| 参数名称      | 参数帮助描述                       |
| ----------- |------------------------------|
| type     | 选择AI大模型（如智普、通义千问...）         |
| model     | 选择模型，默认为GLM-4	                     |
| api-key        | 获取api_key，如果没有该配置，无法与大模型进行对话 |
